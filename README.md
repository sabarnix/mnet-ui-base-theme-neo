# mnet-ui-base-theme-neo

Media.net Theme Neo

## Installation

```
  $ yarn add sabarnix/mnet-ui-base-theme-neo
```

## Usage

```
import { MnetUIBase } from 'mnet-ui-base';
import { neo } from 'mnet-ui-base-theme-neo';

<MnetUIBase theme={neo} />
```
